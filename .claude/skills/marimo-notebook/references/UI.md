# marimo UI Components

## Input Controls
- `mo.ui.text`, `mo.ui.number`, `mo.ui.date` — form controls
- `mo.ui.slider`, `mo.ui.checkbox` — value selectors
- `mo.ui.radio`, `mo.ui.dropdown` — selection widgets

## Data Tools
- `mo.ui.table` — interactive tables
- `mo.ui.data_explorer` — dataframe exploration
- Altair and Plotly chart integration

## Layout
- `mo.ui.tabs` — tabbed layout
- `mo.ui.array` — grouped elements

## External Libraries
Popular alternatives include `ScatterWidget` from `drawdata`, `moutils`, and `wigglystuff`.

## Custom Components
Implement the `_display_` method within a class for personalized interactive UI elements.
